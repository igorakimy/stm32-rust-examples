# Список команд для управления машиной запущенной в симуляторе [Renode](https://github.com/renode/renode)

## Нажать кнопку
```
gpioPortB.breadboardButton_2 Press
```

## Отпустить кнопку
```
gpioPortB.breadboardButton_2 Release
```

## Переключить состояние кнопки
```
gpioPortB.breadboardButton_2 Toggle
```

## Показать текущее состояние кнопки, нажата она или нет
```
gpioPortB.breadboardButton_2 Pressed
```
